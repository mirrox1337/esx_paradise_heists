# esx_paradise_heists
Fleeca Bank Heisty pro FiveM

Potřebné addony:
- ESX

Jazyky:
- Angličtina
- Čeština

Instalace:
1. Běžte do složky resources/[esx]
2. Stáhněte si esx_paradise_heists
3. Přidejte složku do resources/[esx]
4. Přidejte toto do server.cfg: start esx_paradise_heists
